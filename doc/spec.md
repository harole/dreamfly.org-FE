# 项目规范

### 文件、文件夹命名规范

* 使用下划线。


### 代码格式

* 使用2个空格做为缩进。
  Sublime Text 2下强烈建议添加如下选项到项目配置文件的配置对象中：

  ```JavaScript
  "settings":
  {
    // The number of spaces a tab is considered equal to
    "tab_size": 2,

    // Set to true to insert spaces when tab is pressed
    "translate_tabs_to_spaces": true
  }  
  ```


### 缩写规范：使用缩写词前，先来此注册缩写词，以免冲突。

* pub: publish
